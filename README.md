## Сайт pacee4

Описание сайта

**Изменение:** игры и проекты будут перенаправляться со старых каталогов страниц (/games/{game\_name}/, /projects/{project\_name}/) на новые (/game\_{game\_name}/, /{project\_name}/);

**где** {game\_name}, {project\_name} — это заполнители, замененные на исходном сайте.
